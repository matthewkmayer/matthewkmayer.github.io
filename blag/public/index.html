Rusoto RDS walkthrough

Sunday, May 14, 2017
Prerequisites Default VPC (for publically accessibly RDS instances), AWS access keys in some fashion, Rust nightly. Switch to Rust nightly: rustup default nightly. Rocket site Follows the guide at https://rocket.rs/ . Rocket is why nightly Rust is required: Rusoto and Diesel work on stable Rust. Creating a Postgres RDS instance See rusoto-rds/src/main.rs for the full code. In the rusoto-rds directory, run cargo run to create a new database and wait for it to be available.
